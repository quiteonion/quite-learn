# 逻辑或 逻辑与

## 逻辑或

表达为 | ，逻辑或便是 A | B 当前项和后项中有一个正确的时候则这个整体表达为 true。

进阶用法为 || 作用便是在计算机进行运算时 前项已经正确的时候就跳过后项的判定

## 逻辑与

表达为 & ， 逻辑与便是 A & B 当前项与后项全都正确的时候则整体才为 true

进阶用法为 && 作用便是在计算机进行运算的时候，前项已经错误了就跳过后项判断直接为 false
